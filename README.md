# Address Book

With this app you will be able to register user, add other users to your address book or delete them. You can also edit your user data. To use you need to be login.

# Tech:
- ASP MVC
- Entity Framework
- XUnit
- Selenium
- Fluent Validation
- Automapper
- Microsoft Identity

# Overview

## Register page
All inputs are required. Country, Country code and phone code data are from [Json file](./AddressBook/wwwroot/Json/CountryData.json)
